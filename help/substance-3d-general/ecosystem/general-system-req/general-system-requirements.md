+++Linux


Red Hat Enterprise Linux (RHEL) 9.x: The xcb-util-cursor package required by our applications was removedfrom the EPEL 9 repository on 10 May 2024. From this date, the package is supported by the default RHEL 9 repository, coinciding with the release of RHEL 9.4. Customers using versions of RHEL prior to 9.4 will no longer be able to source the xcb-util-cursor package from the EPEL 9 repository after 10 May 2024.





For more information about this package's availability in the default RHEL 9 repository, please refer to:Red Hat Enterprise Linux 9 - Package manifest - AppStream repository



[Red Hat Enterprise Linux 9 - Package manifest - AppStream repository](https://access.redhat.com/documentation/en-us/red_hat_enterprise_linux/9/html-single/package_manifest/index#AppStream-repository)



For more information on this change, please refer to:Red Hat Bugzilla – Bug 2238273,Red Hat Bugzilla – Bug 2212302



[Red Hat Bugzilla – Bug 2238273](https://bugzilla.redhat.com/show_bug.cgi?id=2238273)

[Red Hat Bugzilla – Bug 2212302](https://bugzilla.redhat.com/show_bug.cgi?id=2212302)

## Additional notes

### Unsupported configuration

* Apple computer configurations with eGPUs and user installed GPUs are not supported.
* Windows virtual machines are not supported.
* Windows server operating system is not supported.

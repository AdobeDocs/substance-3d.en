# Thea

Thea Substance™ Converter allows you to create a Thea material from a Substance™ SBSAR file. You can use the parameters exported with the Substance™ file to change the look of the textures and also see the preview of the created Thea material before you save the final result.

Thea Substance™ Converter allows you to create a Thea material from a Substance™ SBSAR file. You can use the parameters exported with the Substance file to change the look of the textures and also see the preview of the created Thea material before you save the final result. You can use any SBSAR file, but with a Substance™ material based on the Metallic/Roughness PBR template you ensure that the created Thea material will also be a physically-based one.

Please visit the Thea rendering docs for more information. [Thea Render Substance Converter](http://thearender.media/docs/fr/thea-substance-converter/menu)

A custom template can be made for exporting textures to Blender.

  1. With the PBR Metal Roughness template selected, click the Duplicate button to make a copy.
  2. Change the Normal type on the copy from DirectX to OpenGL by dragging the option from Converting Maps to the Normal RBG square and selecting "From RBG".
  3. Rename the Preset if desired

Note

Maps that represent data will need to be interpreted correctly. Please see the Color Management page for more information.

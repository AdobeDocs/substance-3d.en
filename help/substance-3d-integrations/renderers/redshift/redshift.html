Redshift

The Redshift material in version 2.5.43 supports the metallic workflow. For rendering with Redshift, you can use maps exported from Substance Painter or the Substance plugin in Maya.

Substance Painter 2020.1 (6.1.0) ships with the Redshift (rsMaterial) Output template as well a legacy output for Redshift 3.

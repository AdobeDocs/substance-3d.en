Ecosystem and Plug-ins

The Substance material sbsar format is supported by all major game and Digital Content Creation (DCC) tools. The format is published from Designer as an .sbsar file. This sbsar file can be loaded into DCC applications and game engines that support Substance materials via a Substance plugin or a native integration.

  • Free Substance materials can be found on the 3D community assets website.
  • Downloadable Substance materials are available via the Substance 3D Assets website as part of the Substance subscription.
  • Substance materials can be made with Substance 3D Designer. (Be sure to check out our Optimization Guidelines. )

For more information on how to use those materials, see:

  • Game Engines
  • 3D Applications
  • Creative Cloud Applications
  • Renderers

Ecosystem and Plug-ins

The Substance material sbsar format is supported by all major game and Digital Content Creation (DCC) tools. The format is published from Substance 3D applications as an .sbsar file. SBSAR files can be loaded into DCC applications and game engines that support Substance materials via a Substance plugin or a native integration.

For more information on how to use those materials, see:

  • Game Engines
  • 3D Applications
  • Creative Cloud Applications
    • After Effects
    • Photoshop
  • Renderers

If you'd like to experiment with SBSAR files, there are many ways to get your own SBSAR materials:

  • Free Substance materials can be found on the 3D community assets website.
  • Downloadable Substance materials are available via the Substance 3D Assets website as part of the Substance subscription.
  • Substance materials with exposable parameters can be made with Substance 3D Designer and Substance 3D Sampler. (Be sure to check out our Optimization Guidelines. )
  • Substance 3D Connector can be used to seamlessly share Substance materials between first and third party applications. For more information, please read our official blog post.

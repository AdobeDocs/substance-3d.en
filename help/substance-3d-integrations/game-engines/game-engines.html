Game Engines

You can import a Substance file (.sbsar) in most major engines via the Substance Plugin.

  • Unity has the substance plugin natively integrated for the 2017 release cycle. For Unity 2018 and higher, the Substance plugin is available in the Unity Asset Store.
  • The Unreal Engine plugin can be found on the Unreal Marketplace.
  • The Lumberyard plugin is natively integrated.
  • The CRYENGINE plugin is natively integrated and maintained by Crytek. For more information, please go to Substance CRYENGINE documentation

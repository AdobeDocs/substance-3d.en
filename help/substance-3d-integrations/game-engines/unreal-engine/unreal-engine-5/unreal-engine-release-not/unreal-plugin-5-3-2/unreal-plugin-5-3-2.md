# Unreal plugin 5.3.2

Released December 4th, 2023

<b>Fixed</b>

* Addressed compatibility issue with the Substance plugin not functioning on Mac for Unreal Engine 5.3.
* Fixed an issue where deselecting a material channel in the Substance plugin for Unreal Engine no longer causes a crash
* Resolved an issue where the 'stone-arc.sbars' material in the plugin displayed only 'Output Size' options under the Input section, instead of showing a full range of modifiable material properties
* Fixed a color shifting issue in the Substance plugin, where imported materials display incorrect, red-shifted normals and off-color base maps.
* Fixed a crash issue in the Substance plugin where using 'Reimport' on a .sbsar file causes the editor to crash.
* Improved input validation in the Substance plugin to prevent out-of-range and invalid entries in key material property fields."
* Fixed a bug where creating a second Graph Instance in the Substance plugin removes all previously loaded presets."
* Fixed a problem where reimporting a Substance Factory Instance removes all existing presets.

# Substance 3D Plugin Default Templates

*Note: This feature is currently in development and it will be available soon.*

With version 5.0.0, we have introduced revamped material template workflow *with numeric outputs supported by the Substance Engine. Material templates allow you to define exactly how you want to configure your Substance material shaders in UE5.*

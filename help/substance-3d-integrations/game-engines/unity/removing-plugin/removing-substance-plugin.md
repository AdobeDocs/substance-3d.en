# Removing Substance Plugin

You can remove the Substance materials before or after deleting the Adobe folder and the Adobe.meta file.

1. Close Unity. Unity needs to be closed in order to remove the plugin. If the project is open and the plugin is removed, it will keep re-importing the engine dll.
1. Using a file explorer, browse to the Asset Folder and delete the Adobe folder and the Adobe.meta file.
1. Restart Unity.

Downloading Substance 3D Plugin in Unity

Asset Store

The Adobe Substance 3D for Unity is available in the Unity Asset Store. The plugin is added to your project in the Assets Folder.

Package Manager

Once you have added and downloaded the plugin via the Asset Store, you can then access the plugin using the Package Manager (Unity 2020.3.27x and higher) under My Assets.

Package Manager Cache

The Asset Store cache can affect which version you see available in the Package Manager/My Assets. If you are not getting the latest version, you may need to delete the Asset Store Cache.

# Unity 2.4.1

## Bug Fixes/Updates:

* When using the HDRP, the metallic value on the generated HDRP Material is set to 0. Fixed it to default to 1.0.
* HDRP Mask output was set to sRGB - Fixed to be disabled sRGB
* Read/write is no longer enabled on Non-substance textures
* Android now properly supports 64bit.
* Fixed Substance Source not loading on Mac OS

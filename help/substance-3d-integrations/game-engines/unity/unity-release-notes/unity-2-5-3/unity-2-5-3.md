# Unity 2.5.3

Released November 4th, 2020

Fixed:

* Crash when changing packing for Specular map
* Specular Map is not packing Smoothness in Alpha channel in Standard 3d Project
* Crash on relaunch of HDRP/HRP project after deleting the Library folder
* Duplicated materials missing assigned shader and material properties
* End of line character changes in scene &amp; prefab files.

Known issues:

* Unable to download assets from Source in Unity consistently. Work around is to download from the [Substance Source website](https://source.substance3d.com/) and manually import the assets.

# Unity 2.4.3

Released Dec 2019

Bug Fixes:

* Fixed: Normal map inverted
* Fixed: Starting Vertical Position issue within Inspector
* Fixed: scrollbar handling in Inspector Graph's "Generated Textures" section. Debug text cleaned up
* Fixed: 'PC, Max, and Linux' spelling error on the Platform Resolution hover text
* Fixed: Error when changing Platform Resolution settings and clicking the Material navigation button
* Fixed: HDRP packing is not correctly handling AO. Mask map green channel is black
* Fixed: (Regression, Mac only) Numerical Outputs / Substances with no valid texture maps crash Unity.
* Fixed: Crash when changing parameters and clicking on another instance of a Substance.
* Fixed: GUI error when clicking on main Substance object, then clicking on Substance graph (in 2019.2+ only)
* Fixed: Unity 2019.3 Error on sbsar file import
* Fixed: Duplicate() function doesn't completely render the Substance unless there is a render call in the script

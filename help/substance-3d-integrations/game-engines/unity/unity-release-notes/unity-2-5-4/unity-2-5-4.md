# Unity 2.5.4

Released December 9th, 2020

Fixed:

* Unable to generate bitcode compatible iOS apps when Substance materials are included in a Unity project

Known issues:

* Unable to download assets from Source in Unity consistently. Work around is to download from the [Substance Source website](https://source.substance3d.com/) and manually import the assets.
* Unity 2020.2 and higher is currently not supported

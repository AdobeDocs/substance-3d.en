# Game Engines

You can import a Substance file (.sbsar) in most major engines via the Substance Plugin.

* **Unity** has the substance plugin natively integrated for the 2017 release cycle. For Unity 2018 and higher, the Substance plugin is available in the [Unity Asset Store](https://assetstore.unity.com/packages/tools/utilities/substance-in-unity-110555).
* The **Unreal Engine** plugin can be found on the [Unreal Marketplace](https://accounts.unrealengine.com/logout?redirectUrl=https%3A%2F%2Fwww.unrealengine.com%2Fmarketplace%2Fsubstance-plugin).
* The **Lumberyard** plugin is natively integrated.
* The **CRYENGINE** plugin is natively integrated and maintained by Crytek. For more information, please go to [Substance CRYENGINE documentation](https://docs.cryengine.com/pages/viewpage.action?pageId=44968527)

For more information, see:

* [Unreal Engine](unreal-engine/unreal-engine.md)

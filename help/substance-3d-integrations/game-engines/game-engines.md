# Game Engines

You can import a Substance file (.sbsar) in most major engines via the Substance Plugin.

* <b>Unity</b> has the substance plugin natively integrated for the 2017 release cycle. For Unity 2018 and higher, the Substance plugin is available in the [Unity Asset Store](https://assetstore.unity.com/packages/tools/utilities/substance-in-unity-110555).
* The <b>Unreal Engine</b> plugin can be found on the [Unreal Marketplace](https://accounts.unrealengine.com/logout?redirectUrl=https%3A%2F%2Fwww.unrealengine.com%2Fmarketplace%2Fsubstance-plugin).
* The <b>Lumberyard</b> plugin is natively integrated.
* The <b>CRYENGINE</b> plugin is natively integrated and maintained by Crytek. For more information, please go to [Substance CRYENGINE documentation](https://docs.cryengine.com/pages/viewpage.action?pageId=44968527)

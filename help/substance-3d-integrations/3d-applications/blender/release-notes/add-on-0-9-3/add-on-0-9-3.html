Add-on 0.9.3

Release notes for Addon version 0.9.3+

  • Add OCIO 2.0 colorspaces labels in the add-on preferences.
  • Add descriptions to the parameter/output factory as tooltips.
  • Add reporting functionality (overall) (removed invasive pop-ups, and replaced them with native blender messages).
  • Update the SRE version label when installing and updating in the addon preferences.
  • Fix Outputsize and randomseed not always available in a substance.

Preferences Section: Added/Updated:

  • Cycles/Eevee Physical Size as an available preset from the default shader drop-down.
  • Controls for Physical Size default projection blend.
  • Changed the support link to point the discord server.
  • Fix display of the correct current SRE version in the addon preferences.
  • Show all the compatible versions of the addon if update is needed.
  • Show array of supported versions instead of first one when SRE missing.

Substance 3D Panel Section: Added/Updated:

  • The Physical Size of a material is displayed (in meters) under the Graph Parameters.
  • Cycles/Eevee Physical Size is available from the shader drop-down.
  • Hide Parameter groups if all parameters are invisible.

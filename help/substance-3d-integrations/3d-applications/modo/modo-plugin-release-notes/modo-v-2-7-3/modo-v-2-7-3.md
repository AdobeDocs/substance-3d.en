# Modo v. 2.7.3

Released April 7th, 2020

Added:

* Supports Modo 14
* Value Outputs in User Channels Support
* Source code available on Adobe Git
* Mac version compatible with Catalina

Fixes:

* Mac version issues caused by Substance Source Send to - removed feature with plans to refactor for a later release

Supports Modo versions 11.0 and higher

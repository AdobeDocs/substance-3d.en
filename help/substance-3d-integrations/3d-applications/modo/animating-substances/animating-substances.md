# Animating Substances

## Animating SubstanceTextures

The advanced parameters can be animated just as any other channel in MODO. If you are using the Substance Engine Time variable ($time) to animate a substance, you can access this specific engine variable on the User Channels for the Substance.   
By animating this channel, you can drive animation if the Substance is built using the $time variable.

3ds Max 2.4.1

Released December 18th, 2020

Added:

  • Upgraded to be compatible with Substance Engine v.8.0.3, supporting new nodes added in Designer 2020.2
  • Edit preset export to export all selected presets (multiple export)

Fixed:

  • The substance_link.dll consuming large number of CPU cycles
  • Assertion error when exporting a Substance preset
  • Crash when creating a preset when there is none
  • Crash with denoiser for Corona renderer

This version is released for 3ds Max 2019, 2020 and 2021

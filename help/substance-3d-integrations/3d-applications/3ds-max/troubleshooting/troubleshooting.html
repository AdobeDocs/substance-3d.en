Troubleshooting

The Scripting Listener can be used to diagnose errors encountered while using the plugin. To open the script listener, navigating to Scripting Menu > Script Listener. When an errors occurs during plugin use, a corresponding error message prints to this  Script Listener window. Visit the official Script Editor documentation for more information.

To report a bug, please join the #3dsmax-plugin channel on the Substance Discord server or visit Adobe communities. Relevant information from the console log and any reproductions steps for the issue can be included in reports. 

Known Issues

  • Replacing an .sbsar that uses a diffuse output with a .sbsar that does not use a diffuse leads to black render due to disconnection of the missing diffuse.
    • This is expected behavior for multi output nodes. Instead of loading these .sbsars using the same node, it is recommended to use different Substance nodes for each.

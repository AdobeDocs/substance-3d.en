# Maya 2.4.0

<b>Added/Updated:</b>

* Upgraded the Substance engine to version 9 in the Maya plugin, enhancing performance and compatibility.

<b>Fixed:</b>

* Fixed a critical issue in Maya 2023 where creating a network with sbsar files from Substance engine version 9 or earlier caused the application to crash, ensuring stable network creation with these files.

Maya 2.4.0 is released for Maya 2020, 2022, and 2023 on Linux, macOS, and Windows

# Maya 2.2.2

Change List:

* Substance 3D Remote Engine updated to 8.4.2
* Added support for Maya 2023
* Added support for Arnold 7.1.0/MtoA 5.1.0 in Maya 2023
* Arnold workflow using 'apply workflow to maps' now correctly identifies 'metalness' map from Painter Arnold template
* If you apply a preset, the resolution dropdowns now update correctly
* 'substanceRunImageWorkflow' command will now return the final shader that's created
* 'substanceNodeApplyWorkflow' command will also return the final shader that's created
* 'substanceUtilityCreateAndLoad' command will now return the substanceNode that it creates

1. 2.2 is released for Maya 2017, 2018, 2019, 2020, 2022, and 2023 on Linux, macOS, and Windows

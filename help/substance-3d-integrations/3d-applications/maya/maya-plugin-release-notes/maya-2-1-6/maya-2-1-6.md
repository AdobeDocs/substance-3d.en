# Maya 2.1.6

Released September 22, 2020

Change List:

* Upgraded to be compatible with Substance Engine v.8.0.2, supporting new nodes added in Designer 2020.2

This version is released for Maya 2017, 2018, 2019 and 2020 on Linux, MacOS and Windows, and Maya LT 2018, 2019 and 2020 on MacOS and Windows

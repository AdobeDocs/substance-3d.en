Maya 2.4.1

Added/Updated:

  • The Maya plugin now utilizes the GPU by default, significantly improving performance and rendering speed.

Fixed:

  • Corrected the link in the About Substance menu of the Maya plugin to redirect to the proper release notes page, now pointing correctly to https://helpx.adobe.com/substance-3d-integrations/3d-applications/maya/maya-plugin-release-notes.html.
  • Resolved an issue in Maya plugin where the displayed resolution value would not update in the UI after resetting a material, ensuring all values including resolution now correctly reset to their default states.

 

Maya 2.4.1 is released for Maya 2020, 2022, and 2023 on Linux, macOS, and Windows

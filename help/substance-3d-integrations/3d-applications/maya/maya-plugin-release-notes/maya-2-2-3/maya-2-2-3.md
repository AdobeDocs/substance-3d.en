# Maya 2.2.3

Change List:

* Added support for Arnold 7.1.0/MtoA 5.1.0 for Maya 2019, 2020 and 2022
* Added a new scripting command, 'substanceNodeGetGraphMetadata', to acquire the Substance graph metadata as an XML string in the script
* Querying an out of bounds graph in scripting calls no longer crashes
* Arnold and MtoA plugins are now installed in a directory using '-' delimiters between semantic version numbers
* Engine resource settings are now appropriately honored, fixing a regression where they were ignored.

1. 2.3 is released for
1. Maya 2017, 2018, 2019, 2020, 2022, and 2023 on Linux, macOS, and Windows
1. Maya LT 2018, 2019, and 2020 on macOS and Windows

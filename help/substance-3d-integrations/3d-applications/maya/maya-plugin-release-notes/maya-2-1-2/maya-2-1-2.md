# Maya 2.1.2

Released June 2, 2020

Today we have released the 2.1.2 version of the plugin.  
  
The only change it contains is:

* Fixed error in Maya LT when creating stingray shader network.

This was also released with Maya 2020.2, and is contained in the Maya installer for that.  
  
It is released for the same platforms, Maya 2017 - 2020 and Maya LT 2018 - 2020

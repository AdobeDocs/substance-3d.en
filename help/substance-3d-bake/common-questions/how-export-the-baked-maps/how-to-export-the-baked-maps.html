How to export the baked maps ?

Alert

Question

How can I export the textures I baked from my mesh to use them in another application ?

Note

Solution : Substance Designer

The output files of the Bakers should be located next to the Substance graph where the mesh is linked to. Right-click and choose "explore" to view the files in the Explorer window of the application.

Note

Solution : Substance Painter

The output files of the Bakers are saved inside the project file (*.spp). To export them it is possible to either :

  • Use the File > Export Textures dialog and choose the export preset "Mesh maps"
  • Right-click on the a baked texture in the Shelf and choosing "Export".
Note

Solution : Substance Automation Toolkit

The output files of the Bakers are saved next to the mesh by default. The output folder can be changed via a specific argument in the command line.

Alert

Question

What is the bit depth of baked textures ?

Note

Solution : Substance Designer

The bit depth depends of the target file format used to save the textures.

Note

Solution : Substance Painter

The baker outputs are generated as 16bits RGBA textures. Then the final bit depth depends of the target file format used to save the textures. 32bits file format will convert the 16bits input information but won't gain more precision.

Height Map from Mesh

The Height Map from mesh baker allows you to create a height map from a high poly mesh.Available in:

  • Painter
  • Designer
  • Automation Toolkit

Parameters

ParameterDescription
Normalization

Defines how the height range of values should be saved down into the texture.

Possible values:

  • Relative to ray distance:
  • Relative to low poly mesh (per UV Tile) (default)
  • Relative to Min/Max (per UV Tile)
  • Manual
Scaling divisor

Define how much the height values should be multiplied or divided.

Only available when the Normalization is set to Manual.

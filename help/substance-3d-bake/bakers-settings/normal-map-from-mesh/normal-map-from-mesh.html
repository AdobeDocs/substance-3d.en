Normal Map from Mesh

The Normal Map from mesh baker allows you to create a tangent space or world space normal map from a high poly mesh.Available in:

  • Substance Painter
  • Substance Designer
  • Substance Automation Toolkit

Parameters

ParameterDescription
Map Type

Controls which type of normal texture the baker should output.

Possible values:

  • World Space
  • Tangent Space (default)

In Substance Painter this parameter cannot be controlled and is set to Tangent Space.

Normal Orientation

Defines the format of the normal texture if the map type parameter is set to Tangent Space.

Possible values:

  • OpenGL
  • DirectX (default)

In Substance Painter this parameter is controlled by the project settings.

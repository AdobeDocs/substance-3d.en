Convert UV to SVG

The Convert UV to SVG baker transforms the low-poly mesh UVs into a vector graphic file. This vector graphic file can be used to create masks.

Available in :

  • Substance Designer
  • Substance Automation Toolkit

Parameters

ParameterDescription
PaddingControls how much geometric padding to add to the SVG shapes.
Color Mode

Defines how the SVG shapes should be colored.

Possible values:

  • Random: each UV shell is colored by a random color.
  • HUE Shift: each UV shell is colored by a unique hue value.
  • Grayscale: each UV shell is colored by a unique grayscale value.
  • Uniform color: all UV shells are colored by a 50% grey value.
  • Material ID Color: UV Shells are colored by the material color defined in the Scene View.

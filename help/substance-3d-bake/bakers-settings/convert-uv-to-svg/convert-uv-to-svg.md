# Convert UV to SVG

The Convert UV to SVG baker transforms the low-poly mesh UVs into a vector graphic file. This vector graphic file can be used to create masks.

**Available in :**

* Substance Designer
* Substance Automation Toolkit

## Parameters

| *Parameter* | *Description* |
| --- | --- |
| **Padding** | Controls how much geometric padding to add to the SVG shapes. |
| **Color Mode** | Defines how the SVG shapes should be colored.Possible values:<ul data-preserve-html="true"><li data-preserve-html="true"><strong>Random:</strong> each UV shell is colored by a random color.</li><li data-preserve-html="true"><strong>HUE Shift:</strong> each UV shell is colored by a unique hue value.</li><li data-preserve-html="true"><strong>Grayscale:</strong> each UV shell is colored by a unique grayscale value.</li><li data-preserve-html="true"><strong>Uniform color:</strong> all UV shells are colored by a 50% grey value.</li><li data-preserve-html="true"><strong>Material ID Color</strong>: UV Shells are colored by the material color defined in the Scene View.</li></ul> |

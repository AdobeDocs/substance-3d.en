# Opacity Mask from Mesh

The Opacity Mask from mesh baker compute a texture where missed rays are converted into a grayscale binary image. This texture is generally used as an opacity map.

The texture properties are defined as:

* Black values represent empty areas.
* White values represent convex areas.

**Available in:**

* Painter
* Designer
* Automation Toolkit

## Parameters:

This baker has no specific parameters.

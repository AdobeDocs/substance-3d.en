World Space Normals

The World Space normals baker allows to save the mesh normal, tangent and binormals into a texture.

Available in :

  • Substance Designer
  • Substance Automation Toolkit

Parameters

ParameterDescription
Baking Type

Defines which type of computation the baker will perform.

Possible values:

  • Normal (default)
  • Tangent
  • Binormal
Normal Map

Path to the input normal texture that will be used during the computation to add details.

Normal Orientation

Defines the normal format of the input texture if Baking Type is set to Normal.

Possible values:

  • OpenGL
  • DirectX (default)

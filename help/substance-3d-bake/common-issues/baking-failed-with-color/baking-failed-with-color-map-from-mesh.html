Baking failed with Color Map from Mesh

Alert

Issue

Possible error message:

[ Baking ] Baking failed (Color Map from Mesh)
Could not find vertex colors

Note

Explanation

The default settings for the Color Map from Mesh is to bake the high-poly mesh vertex colors into a texture based on the mesh UVs. However it is often the case where the high-poly mesh doesn't have any vertex colors information. Therefore the baker can't write information that don't exist.

Note

Solution

Different solutions are available to avoid this error message :

  • Use an high-poly mesh that has vertex colors
  • Set the Color Map from Mesh baker with different settings
  • Don't use the Color Map from Mesh baker if you don't need it
